# CashFlow Tracker 💰

CashFlow Tracker adalah aplikasi web manajemen keuangan pribadi yang modern dan aman, dirancang untuk membantu Anda mencatat pemasukan dan pengeluaran dengan mudah. Aplikasi ini berjalan sepenuhnya di browser Anda dengan fokus pada privasi data, menggunakan penyimpanan lokal (IndexedDB) sehingga tidak ada data keuangan yang dikirim ke server.


## Teknologi yang Digunakan 🛠️

- **Framework**: Next.js 15 (App Router) & React 19
- **Bahasa**: TypeScript
- **Styling**: Vanilla CSS dengan custom properties (CSS Variables)
- **Database**: IndexedDB (via Dexie.js)
- **Keamanan**: Web Crypto API native browser

## Cara Menjalankan Secara Lokal 🚀

1. Install dependensi:
   ```bash
   npm install
   ```
2. Jalankan development server:
   ```bash
   npm run dev
   ```
3. Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
